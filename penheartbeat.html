pen heartbeat

Nikos Antonio Kourous Vázquez

A single smooth motion traces across the page. It breathes once — no interruptions, no hesitations. This is a baseline — a first heartbeat.

But in later iterations, something is wrong. The same motion, now jittered and fragmented. As if memory or intention had misfired.
